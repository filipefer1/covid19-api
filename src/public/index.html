Covid19 Brasil API

Api desenvolvida para facilitar a utilização dos dados da covid 19 no Brasil.

Endpoints

GET - https://localhost:3333/casos

Retorna todos os casos de covid no Brasil.

GET - https://localhost:3333/casos/{data}

Retorna todos os casos de covid na data especificada.

Exemplo: https://localhost:3333/casos/2020-03-27

GET - https://localhost:3333/{UF do estado}

Retorna todos os casos de covid no estado especificado.

Exemplo: https://localhost:3333/df

GET - https://localhost:3333/{UF do estado}/{data}

Retorna todos os casos de covid no estado e na data especificado.

Exemplo: https://localhost:3333/df/2020-03-27

Parâmetros

UF do estado: Utilize a sigla da unidade da federação para retornar os dados do estado.

Possíveis erros:

UF especificada não possui dois caracteres.

400 Bad Request

{ "message" : "Tamanho da UF inválido." }

UF especificada não existe.

400 Bad Request

{ "message" : "UF inexistente." }

Data: Para as requisições que necessitem de data utilize o formato 'YYYY-MM-DD' e insira as datas a partir de 2020-02-26 (primeiro caso de covid no Brasil).

Possíveis erros:

Data não encontrada ou fora do formato YYYY-MM-DD

400 Bad Request

{ "message" : "Data não encontrada. Utilize datas a partir de 2020-02-26 e o formato 'YYYY-MM-DD' " }
